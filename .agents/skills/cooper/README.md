# Cooper Skill 使用说明

## 功能简介

Cooper skill 让你可以通过自然语言与Cooper进行交互，轻松管理团队空间、知识库、搜索文档内容、创建文件夹和文档、下载文件、查看最近访问的资源等。

## 核心功能

### 1. 🏢 团队空间管理
- 列出我拥有或参与的团队空间
- 浏览空间目录结构（含个人空间）
- 在空间中创建文件夹进行分类整理
- 在空间或文件夹中创建文档（支持 markdown）

### 2. 📚 知识库管理
- 列出我的知识库
- 浏览知识库目录结构
- 创建新的知识库页面
- 管理知识库页面标签（查看、添加）

### 3. 📄 文档操作
- 读取文档内容
- 支持普通文档和知识库文档
- 读取表格指定单元格范围（如 Sheet1!A1:D10）

### 4. 📎 文件管理
- 从空间中下载附件文件

### 5. 🔍 搜索功能
- 全文搜索所有内容
- 快速找到需要的文档

### 6. 🏷️ 标签管理
- 查询知识库已有标签列表
- 查看页面已有标签（含安全等级标签和自定义标签）
- 给知识库页面添加标签（支持自动创建新标签）

### 6. 📊 最近资源
- 查看最近访问的文档
- 查看最近编辑的文档
- 查看与我分享的文档
- 按类型过滤（文档、表格、PPT、文件等）

## 快速开始

### 配置要求

1. 确保已安装 mcporter：
```bash
npm install -g mcporter
```

2. 配置 Cooper MCP 服务（配置文件：`~/config/mcporter.json`）：
```json
{
  "mcpServers": {
    "Cooper": {
      "baseUrl": "http://10.88.128.45/cooper_mcp/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_TOKEN"
      }
    }
  }
}
```

3. 从 MCP Hub 获取 Token：https://mcphub.intra.xiaojukeji.com/

## 支持的操作

### 团队空间操作
- `listCooperSpaces` - 列出团队空间（我拥有的/我参与的）
- `getCooperSpaceDirectory` - 获取空间目录结构（含个人空间）
- `createCooperDirectory` - 在空间中创建文件夹
- `createCooperDocument` - 在空间中创建文档（markdown 格式）
- `downloadCooperFile` - 下载空间中的文件

### 知识库操作
- `listKnowledgeBases` - 列出知识库
- `getKnowledgeDirectory` - 获取知识库目录
- `createKnowledgePage` - 创建知识库页面
- `getKnowledgeBaseTags` - 查询知识库已有标签
- `getPageTags` - 查询页面标签
- `addPageTag` - 给页面添加标签（支持自动创建新标签）

### 文档操作
- `readContent` - 读取文档内容（支持表格范围读取）

### 搜索和发现
- `search` - 搜索内容
- `listRecent` - 查看最近资源（支持按类型过滤）

## 参数说明

### spaceId
- `"0"` - 个人空间
- 其他值 - 团队空间 ID（从 `listCooperSpaces` 获取）

### knowledgeId / spaceId (知识库)
知识库的唯一标识符，通常是数字 ID

### resourceId
文档、页面或文件的唯一标识符

### parentId
父目录 ID，根目录使用 `"0"`

### appId
- `2` - Cooper 普通文档
- `4` - 知识库文档

### range (readContent)
表格单元格范围，格式：`SheetName!A1:B2`（如 `Sheet1!A1:D10`），仅对表格类型文档有效

### 标签参数 (addPageTag / getPageTags / getKnowledgeBaseTags)
- `knowledgeId` — 知识库ID，用于查询知识库的全部标签
- `pageId` — 知识库页面资源ID，用于查看或添加页面标签
- `tagId` — 标签ID，已有标签传入ID；新标签传空字符串 `""`
- `tagName` — 标签名称，当 `tagId` 为空时系统会自动创建新标签
- 标签仅支持知识库页面，不支持 Cooper 空间文档

### queryType (listRecent)
- `1` - 最近访问
- `2` - 最近编辑
- `3` - 与我分享
- `4` - 最近互动
- `5` - DC 消息

### type (listRecent)
- `all` - 全部类型
- `coo_doc` - 文档
- `coo_sheet` - 表格
- `coo_ppt` - PPT
- `coo_file` - 文件
- `flow` - 流程图
- `mind` - 思维导图

### queryType (listRecent)
- `1` - 最近访问
- `2` - 最近编辑
- `3` - 与我分享
- `4` - 最近互动
- `5` - DC 消息

## 使用技巧

1. **先列出，再浏览**：使用 `listKnowledgeBases` 查看所有知识库，然后选择感兴趣的进行浏览

2. **善用搜索**：不确定文档在哪个知识库时，直接搜索关键词

3. **查看最近**：忘记文档位置时，查看最近访问列表快速定位

4. **指定类型**：读取文档时，正确指定 appId（普通文档用 2，知识库用 4）

## 故障排查

### 问题 1：无法连接到 Cooper
- 检查网络连接
- 验证配置文件路径和格式
- 确认 Token 是否有效

### 问题 2：返回 401/403 错误
- Token 可能已过期，重新从 MCP Hub 获取
- 确认 Token 格式为 `Bearer <token>`

### 问题 3：找不到文档
- 检查 resourceId 是否正确
- 确认 appId 参数（2 或 4）
- 验证是否有访问权限

### 问题 4：搜索无结果
- 尝试不同的关键词
- 检查搜索范围和权限
- 使用更通用的搜索词

## 注意事项

1. **权限要求**：只能访问有权限的知识库和文档
2. **Token 安全**：妥善保管 Token，不要泄露
3. **网络环境**：需要在内网环境访问
4. **数据修改**：创建操作会真实创建文档，请谨慎操作
